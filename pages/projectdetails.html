Overview

Snakewall is a modern twist on the classic Snake game, re-imagined as a dynamic Live Wallpaper. Built for Android, it features smooth animations, adaptive design, and minimalist visuals.
Technical

The app is developed using Java in Android Studio, leveraging the Android Live Wallpaper Engine and canvas API.

- The wallpaper adapts to different screen sizes and orientations, ensuring a consistent user experience across devices.

- The main board is an odd number grid.

- Snake is a linked list representing coordinates on the grid.

The Challenge

Adapting the classic Snake game into a Live Wallpaper presented unique challenges, particularly in maintaining performance and responsiveness. The dynamic nature of live wallpapers requires efficient resource management to ensure smooth animations without draining battery life.
